## Description of Component

The Zero-X Evaluation Engine analyses the constructed models to verify schedule feasibility and to evaluate performance against the project's Zero-X goals, such as waste reduction, circularity and resilience. It provides the circularity and sustainability specifications that are taken into account when building each pilot's formal model.
