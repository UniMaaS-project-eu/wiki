## Description of Component

Serves as the semantic backbone using an MSC-based ontology to enable the Formal Models Engine per pilot. Central knowledge graphs contain data shared by the data space participants regarding their manufacturing processes. Details about the MSC ontology and requirements regarding data interoperability are described in detail in section 3.
