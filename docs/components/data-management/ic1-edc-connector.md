## Description of Component

A UniMaaS-based extension of the Eclipse Dataspace Connectors (EDC) to enable sovereign, trusted cross-company data exchange. Specifically, the EDC connector has been extended to support data transfers of RDF datasets to serve the DMS central knowledge graph. These extensions are deployed in conjunction with the implementation of IAM & Data Governance.
